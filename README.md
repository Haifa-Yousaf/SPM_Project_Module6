Communication & Notifications Module
#Overview

The Communication & Notifications Module is a core component of the National Freelance and Skill Verification Platform. It enables secure, real-time interaction between freelancers and clients, ensuring efficient collaboration throughout project lifecycles.
This module centralizes all communication within the platform, reducing reliance on external tools and improving transparency, productivity, and user experience.

#Features
Real-time Messaging
Instant one-to-one communication between freelancers and clients
Group Chats
Support for team-based communication involving multiple users
File Sharing
Upload, send, and download files within conversations
Notifications System
Real-time alerts for messages, file updates, and activity
Meeting Integration
Generate and share meeting links via Zoom / Google Meet
Message History
Persistent storage and retrieval of all conversations
Search Functionality
Search messages and files using keywords

#Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js / Python
Database: PostgreSQL
Real-time Communication: WebSockets
APIs: Zoom API, Google Meet integration
Version Control: GitHub
